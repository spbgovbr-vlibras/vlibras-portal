Apresentação¶

O VLibras Desktop é um recurso de acessibilidade desenvolvido para tornar computadores acessíveis para pessoas surdas. Com a tradução automática de Português Brasileiro para a Língua Brasileira de Sinais (LIBRAS), essa ferramenta permite que usuários surdos sejam capazes de consumir conteúdos de texto em qualquer website ou arquivo pessoal.

Este manual tem como objetivo apresentar os procedimentos para a instalação da ferramenta VLibras Desktop em computadores com a distribuição Windows.
