Propor remoção de sinal¶

A remoção de sinal pode ser proposta por todos usuários, visto que todos os usuários possuem o perfil “COLABORADOR”. Os passos para propor uma remoção são:

  1. Na aba “REMOVER SINAL” o usuário poderá pesquisar o sinal que deseja propor a remoção no campo “PESQUISAR SINAL”.

2. Após pesquisar o sinal, aparecerá o sinal que ele deseja propor a remoção. Para propor a remoção, é necessário clicar no botão “VER COMPLETA”.

3. Irá aparecer uma janela com todas as informações do sinal, então para propor a remoção de um sinal, o usuário precisará clicar no botão “REMOVER SINAL”.
